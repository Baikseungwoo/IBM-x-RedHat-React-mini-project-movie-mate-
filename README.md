# IBM-x-RedHat-React-mini-project-movie-mate-
This repo is for react mini project in IBM x RedHat bootcamp.

# 📌 프로젝트 진행 가이드

## 🚀 작업 시작 방법

1. 레포지토리 클론
   - git clone <레포지토리 URL>
   - cd <프로젝트 폴더>

2. 원격 브랜치 가져오기
   - git fetch

3. 원격 브랜치 확인
   - git branch -r

4. 본인 브랜치로 이동
   - git switch <본인-브랜치명>

5. 맡은 파트 구현 진행

---

## 📂 파일 생성 관련

- 기본적으로 필요한 파일들은 미리 생성되어 있음
- 추가 구현에 필요한 파일이 있다면 자유롭게 생성 가능
- 효율적인 개발을 위해 구조 변경 및 파일 추가 가능

---

## ⚠️ 주의사항
